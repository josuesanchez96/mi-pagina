¿Qué es WebRTC?

WebRTC, es un proyecto de código abierto – promovido por Google Chrome, Mozilla Firefox y otros – que permite comunicaciones en tiempo real sin plug-ins a través de una API Javascript. Facilita las aplicaciones de llamadas de voz, video, chat y compartimiento de archivos entre navegadores. Utiliza un Servidor de Conferencias Web que en conjunto con un Servidor STUN es requerido para proveer la página inicial y sincronizar las conexiones entre dos nodos de este tipo.

¿Qué es Socket.io?

Socket.IO es una biblioteca de JavaScript para crear aplicaciones web en tiempo real basadas en eventos. Permite la comunicación bidireccional entre clientes y servidores mediante WebSockets y tecnologías de respaldo.

¿Cómo funciona Socket.IO?

Socket.IO funciona sobre los protocolos HTTP y WebSocket para proporcionar:

  • Comunicación en tiempo real: mensajería de baja latencia entre cliente y servidor.
  • Arquitectura basada en eventos: usted emite y escucha eventos en ambos extremos.
  • Reconexión automática: lógica incorporada para recuperarse de desconexiones temporales.
  • Mecanismos de respaldo: se recurre al sondeo largo si los WebSockets no están disponibles.
